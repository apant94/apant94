### Обо мне
Мечтаю участвовать в воплощении классных и полезных проектов и улучшать их. 
Во-первых, как бывший владелец своего бренда одежды, я люблю делать что-то красивое и практичное своими руками, а во-вторых, я уже выводила свой бренд в онлайн плоскость.
После такого опыта я стала фанатом неограниченных возможностей веб-разработки. Самое главное, что больше всего мне нравится видеть результат своей работы, поэтому frontend - мой идеальный мэтч.

На данный момент разрабатываю 2 приложения:  
1. Таймер Duellio для клуба переговоров  
2. Тайм-трекер EasyHrs для определения эффективности выполнения разных задач сотрудниками
